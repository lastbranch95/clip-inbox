# Clip Inbox

Clip Inbox は、気になったURLをあとで見返すためのシンプルなクリップ保存アプリです。

YouTube、記事、X、買い物候補、技術メモなどを一か所に保存し、あとから検索・整理・掘り返しができます。

## 主な機能

* URLの保存
* 保存理由のメモ
* タグ管理
* 未整理 / 視聴済み / 放置 / 期限切れ / ゴミ箱フィルター
* YouTubeサムネイル表示
* 今日の1本の自動提案
* 視聴回数の記録
* あとで見る期限の設定
* Private表示モード
* JSONバックアップ / 読み込み
* スマホのホーム画面追加に対応

## 使い方

1. 右下の「＋」を押す
2. URLを貼る
3. 必要なら保存理由・タグを入力する
4. 保存する

保存したクリップは一覧に表示されます。

三点メニューから、詳細確認、編集、URLコピー、削除、タイトル編集などができます。

## バックアップ

設定画面からJSONバックアップができます。

* JSONをコピー
* JSON書き出し
* JSON読み込み
* JSON貼り付け読み込み

スマホでも扱いやすいように、JSONは大きなテキスト欄で表示できます。

## Privateモード

Privateクリップは通常表示では隠れます。

設定画面からパスコードを入力すると、非表示クリップを表示できます。

## 注意

このアプリのデータはブラウザ内の IndexedDB に保存されます。

ブラウザのデータ削除、端末変更、キャッシュ削除などで消える可能性があるため、必要に応じてJSONバックアップを取ってください。
