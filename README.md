# Clip Inbox

Clip Inbox は、気になったURLをあとで見返すためのシンプルなクリップ保存アプリです。

YouTube、記事、X、買い物候補、技術メモなどを一か所に保存し、あとから検索・整理・掘り返しができます。

スマホのホーム画面に追加して、PWA風に使うこともできます。

## 主な機能

* URLの保存
* 保存理由のメモ
* タグ管理
* お気に入り登録
* 未整理 / 視聴済み / 放置 / 期限切れ / タグなし / お気に入り / ゴミ箱フィルター
* YouTubeサムネイル表示
* YouTube動画IDベースの重複チェック
* 今日の1本の自動提案
* 今日の1本の再抽選
* 視聴回数の記録
* あとで見る期限の設定
* Private表示モード
* JSONバックアップ / 読み込み
* スマホのホーム画面追加に対応

## 使い方

1. 右下の「＋」を押す
2. URLを貼る
3. 必要なら保存理由・タグを入力する
4. 保存する

保存したクリップは一覧に表示されます。

三点メニューから、詳細確認、編集、お気に入り切替、URLコピー、削除、タイトル編集などができます。

## 今日の1本

保存したクリップの中から、あとで見返す候補を自動で表示します。

未視聴、保存から日数が経っているもの、保存理由やタグがあるもの、お気に入りのものが出やすくなります。

ヘッダーの掘り返しボタンを押すと、今日の1本を再抽選できます。

## お気に入り

重要なクリップや、あとで見返したいクリップはお気に入りにできます。

お気に入りにしたクリップは一覧上で「★」が表示され、お気に入りフィルターから絞り込めます。

## バックアップ

設定画面からJSONバックアップができます。

* JSONをコピー
* JSON書き出し
* JSON読み込み
* JSON貼り付け読み込み

スマホでも扱いやすいように、JSONは大きなテキスト欄で表示できます。

## Privateモード

Privateクリップは通常表示では隠れます。

設定画面からパスコードを入力すると、非表示クリップを表示できます。

## データ保存について

このアプリのデータは、ブラウザ内の IndexedDB に保存されます。

サーバーには保存されません。

ブラウザのデータ削除、端末変更、キャッシュ削除などで消える可能性があるため、必要に応じてJSONバックアップを取ってください。

## 注意

PrivateクリップもJSONバックアップには含まれます。

バックアップファイルを共有する場合は、中身を確認してから扱ってください。
